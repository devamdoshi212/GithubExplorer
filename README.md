
# Github Explorer

## Deployed Link

https://github-explorer-vert.vercel.app/

## Clone Project 

##### Step 1

```bash
  git clone https://github.com/devamdoshi212/GithubExplorer
```

##### Step 2

```bash
  cd GithubExplorer
```

##### Step 3

```bash
  npm install
```

##### Step 4 

Run Project

```bash
  npm run dev
```
